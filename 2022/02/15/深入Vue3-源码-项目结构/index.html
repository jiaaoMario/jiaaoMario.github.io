深入Vue3 源码 - 项目结构

开始

在开始学习源码时，尽量不要直接关注核心逻辑。因为大型开源项目源码的复杂程度和边界逻辑往往是难以想象的，很有可能尝试阅读几分钟后便会沉入代码的汪洋大海之中，从而快速实现从入门到放弃。
在接下来阅读 Vue3 源码之前，我们会首先站在宏观的角度，对其基础架构进行学习。

vuejs/core 目录

package.json

!

在前端项目中，package.json 往往是整个架构的目录，package.json 不仅仅告诉我们该项目的第三方依赖，如devDependencies、dependencies、peerDependencies。还可以通过 npm script 配置的脚本命令了解项目的完整生命周期脚本。

在下图 npm scripts 中，定义了许多任务指令，基础的像 generate changelog、代码风格校验等等，在这里我们暂时只关注核心的构建指令，如dev、build、size、release、dev-esm等。
